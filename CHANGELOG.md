# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-06-16

### Added

- Database-backed capture of outgoing Laravel mail via the `MessageSending` event.
- Web inbox at a configurable path with HTML, text, headers, and attachment metadata views.
- Gate and middleware configuration for access control.
- Publishable config, views, and migrations.
- Orchestra Testbench feature tests.

### Changed

- Package prepared for public release on Packagist.

## [1.0.0] - 2026-06-09

### Added

- Initial release as an internal path package.
